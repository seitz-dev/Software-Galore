# Software Galore

[Steam Workshop Link](https://steamcommunity.com/sharedfiles/filedetails/?id=3702052349)

**Software Galore** is a large, comprehensive data expansion mod for *Software Inc. (Beta 1.8)* that adds 30 entirely new software types into the game. 

Tired of building the same Operating Systems, Games, and 2D Editors? Software Galore expands the technology tree with a hyper-detailed journey from the early days of 1980s mainframe terminals to speculative 2035 sci-fi tech like quantum computing and neural interfaces.

This mod aims to bring back that feeling of old Software Inc where there were plenty of expansion mods, all adding to the novelty and the complexity of the game.

## 🚀 Features

*   **30 Brand-New Software Industries:** Conquer everything from CAD Software and POS Systems in the 80s to Ride-Sharing Apps and Streaming Services in the 2010s.
*   **300+ Deep, Interdependent Features:** Every software type contains 11-14 meticulously balanced features structured hierarchically. They seamlessly span all specializations (`System`, `Network`, `2D`, `3D`, and `Audio`).
*   **Historical Accuracy:** Software unlocks dynamically throughout the decades. Start your company in 1980 by developing Terminal Emulators or Database Systems, pivot to Web Browsers and Email Clients in the 90s, and ride the dot-com wave into Social Media and E-Commerce.
*   **Speculative Sci-Fi Late Game:** The technology tree extends far beyond the modern day. By 2030-2035, you'll be researching *Quantum Compilers*, *Holographic Telepresence*, *Nanobot Blood Analysis*, and *Direct Brain-Computer Interfaces*.
*   **Beta 1.8 Fully Compliant:** Built specifically for the strict data parser of Beta 1.8. It correctly implements `Submarkets`, `CodeArt` math ratios, `Iterative` sequencing, and dynamic `NameGenerators` for AI competitors.
*   **Complete Sandbox Freedom:** All secondary and tertiary specializations are set to `Optional`, meaning you can choose to build a basic local app on a cheap 1980s OS, or wait to build a cutting-edge version that utilizes 3D and Network features on a high-end modern OS.

## ⚖️ Balancing Details
The mod honors the base game's progression curve. Level 1 and 2 features are relatively cheap and can be built by small teams. However, the Level 3 "Sci-Fi" endgame features (unlocking 2025-2035) require astronomical amounts of Code and Art, demanding massive teams of highly educated, specialized engineers to complete.
